# iCook

O iCook é uma plataforma que permite recomendar receitas de acordo com o gosto do cliente a partir dos ingredientes que ele tem em casa. Nosso produto, por meio de um algoritmo de AI, sugere ao usuário listas de compras baseadas nos ingredientes faltantes para concluir a receita desejada.
